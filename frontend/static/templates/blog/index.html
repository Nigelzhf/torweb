JavaScript 创建正则表达式有两种方式： 使用正则表达式字面量：var re = /abc/; 调用 RegExp 对象的构造函数：var re = new RegExp("abc"); 什么时候使用构造函数？ 当正则表达式的模式会变发生改变，或者需要从其他地方得到时（例如用户输入），适合使用构造函数的方式。
